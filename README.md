# apfe-cli

Alipay+ Sub-app ToolKit

## Installation

Prerequisites: [Node.js](https://nodejs.org/en/) version 8+, npm version 5+ and [Git](https://git-scm.com/).

```bash
$ npm i -g apfe-cli
```
